<h2 id="Sobre">:computer: Sobre o projeto</h2>
<p>
  Farmácia - API REST: Este é um projeto back-end voltado para uma farmácia, com ele você pode gerenciar os remédios e as categorias desses remédios.
</p>

<p>
  Esse projeto foi um exercício de revisão da <a href="https://brazil.generation.org/">Generation Brasil</a> para que os alunos pudessem reforçar o conhecimentos das tecnologias de back-end.
</p>

<h2 id="Funcionalidades">⚙️ Funcionalidades</h2>

<h3>Remédio:</h3>

  - [x] Cadastrar remédio
  - [x] Consultar todos os remédios
  - [x] Consultar remédios por nome parecido
  - [x] Consultar remédio através de um ID
  - [x] Alterar cadastro de remédio
  - [x] Deletar cadastro de remédio

<h3>Categoria:</h3>

  - [x] Cadastrar categoria
  - [x] Consultar todas as categorias
  - [x] Consultar categorias por nome parecido
  - [x] Consultar categoria através de um ID
  - [x] Alterar cadastro de catgoria
  - [x] Deletar cadastro de categoria
